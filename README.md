# georgethiel

My personal knowledge base on AI — things I use and have built.

## What's here

- **[`local-ai/`](local-ai/)** — my local AI stack setup: running Ollama +
  Open WebUI on a Mac, hardware notes, model compatibility, and prompts.

- **[`claude-code/skills/`](claude-code/skills/)** — six standalone, downloadable
  Claude Code skills (GIT, MAP, REPORT, REVIEW, COUNCIL, DEBATE). Copy a folder
  into your project's `.claude/skills/` and go.

- **[`claude-code/harness/`](claude-code/harness/)** — the full workflow harness
  those skills came from: hooks, skills, `CLAUDE.md` rules, profiles, and an
  installer that sets up a disciplined build pipeline in a target repo.

Start in whichever tree matches what you're after — each has its own README.
